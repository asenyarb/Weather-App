Add city

To add city to your profile use " *site name*/api/add-city/?city_name=*city name*&code=*country code* ", where city_name - city name and code - country code (ISO 3166-2 standart)
Code value is optional.

Delete city

To delete city from your profile use " *site name*/api/delete-city/?city_name=*city name*&code=*country code* ", where city_name - city name and code - country code (ISO 3166-2 standart)
Code value is optional. If no code value was specified, all cities with specified city name will be deleted
